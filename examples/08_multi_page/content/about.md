# Multi-Page Demo ::title

This app has three pages sharing one `AppState`.

::: frame panel

Every page inherits styles from `_app.md`. Navigation is auto-generated.

:::

## Features

- **Headings** and inline formatting
- **Styled text** via `::key` suffixes
- **Tables** with formatted cells
- **Containers** with frames and colored blockquotes

**Current user:** [display](name)

> Built entirely from markdown files. ::success
